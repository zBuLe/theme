Welcome to the Docs

This is a demonstration of the premium, remote-ready Jekyll documentation theme layout. Notice how the layout responds to resizing the browser window.

Installation

To get started, you will need to install the package via npm or yarn. Ensure you have the latest version of Node installed.

npm install premium-jekyll-theme
Note: This theme requires Jekyll 4.0 or higher to take advantage of the latest build optimizations.

Configuration

Once installed, you can configure the theme in your _config.yml file. Add the following lines to enable the remote theme capabilities.

remote_theme: "your-github-username/premium-jekyll-theme"
plugins:
  - jekyll-remote-theme

Advanced Setup

If you want to customize the search indices or the default color variables, you can override the defaults by creating a local assets/css/style.css.

Writing JavaScript

The theme includes a custom code block feature that automatically detects the language and adds a copy-to-clipboard button.

function greetUser(name) {
  console.log(`Hello, ${name}! Welcome to your new theme.`);
}

greetUser("Developer");
